Dicionário de Tags HTML

<html>

Define o início e o fim de um documento HTML.

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Exemplo de HTML Básico</title>
</head>
<body>
  <h1>Olá, mundo!</h1>
  <p>Este é um exemplo básico de um arquivo HTML.</p>
</body>
</html>

<ul> e <li>

Define uma lista não ordenada.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
  • Item 1
  • Item 2
  • Item 3

<ol> e <li>

Define uma lista ordenada.

<ol>
  <li>Primeiro item</li>
  <li>Segundo item</li>
  <li>Terceiro item</li>
</ol>
  1. Primeiro item
  2. Segundo item
  3. Terceiro item
